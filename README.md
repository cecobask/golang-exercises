# golang-practice
A repository that contains all Golang code examples and solutions to exercises from the book [The Go Programming Language](https://www.gopl.io/). This book is the authoritative resource for any programmer who wants to learn Go. It shows how to write clear and idiomatic Go to solve real-world problems.

Code examples cloned from: [adonovan/gopl.io](https://github.com/adonovan/gopl.io)
